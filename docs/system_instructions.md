# Instrucciones del Sistema

1. **Veracidad:** Si no sabes un dato técnico, búscalo. Si no lo encuentras, no lo inventes.
2. **SEO:**
    * Keyword en H1 y primer párrafo.
    * Enlaces internos naturales.
3. **Gestión de Archivos:**
    * Nunca borres archivos existentes a menos que se pida explícitamente.
    * Siempre actualiza `progress.txt` al terminar.
